# 내 펫 사진 만들기

우리 열 명 펫이 한 화면에 나란히 붙습니다. **크기와 배경이 제각각이면 벽이 지저분해집니다.**
그래서 아래 형식을 지킵니다. 동물만 바꾸면 됩니다.

---

## 1. 프롬프트 - 이 문장을 그대로 복사해서 쓰세요

`[동물]` 자리에만 원하는 동물을 넣습니다.

```
정사각형 1:1 비율로 만들어 주세요.

[동물] 한 마리가 화면 정중앙에 있고, 몸 전체가 다 보이게 해주세요.
배경은 완전한 흰색 단색입니다. 무늬, 그림자, 바닥선 없이 깨끗한 흰색만 있어야 합니다.
동물 주위에 여백을 넉넉히 두고, 몸이 화면 가장자리에 닿지 않게 해주세요.
글자, 로고, 서명은 넣지 마세요.

사진처럼 사실적으로 그려 주세요.
```

**동물 예시**: 바위너구리, 고양이, 코알라, 카피바라, 펭귄, 해달, 알파카, 문어, 앵무새, 두꺼비

## 2. 화풍은 마지막 줄만 바꾸면 됩니다

| 원하는 느낌 | 마지막 줄에 이렇게 |
|---|---|
| 사진 같은 | `사진처럼 사실적으로 그려 주세요.` |
| 그림책 같은 | `수채화 느낌의 부드러운 일러스트로 그려 주세요.` |
| 게임 도트 | `8비트 픽셀 아트로, 픽셀이 또렷하게 보이게 그려 주세요.` |
| 스티커 같은 | `굵은 검은 테두리가 있는 단순한 스티커 그림으로 그려 주세요.` |

## 3. 이렇게 나오면 성공입니다

```
   [                    ]     정사각형이다
   [      (동물)        ]     동물이 가운데 있고 전신이 다 보인다
   [                    ]     배경이 흰색 하나다
                              가장자리에 몸이 안 닿는다
                              글자가 없다
```

## 4. 이러면 다시 만듭니다

- **가로가 더 길다** -> "정사각형 1:1" 을 다시 강조해서 요청합니다.
- **머리만 크게 나왔다** -> "몸 전체가 다 보이게" 를 다시 강조합니다.
- **배경에 풀밭이나 방이 있다** -> "배경은 완전한 흰색 단색, 아무것도 없이" 로 다시 요청합니다.
- **몸이 잘렸다** -> "여백을 넉넉히, 가장자리에 닿지 않게" 를 다시 강조합니다.
- **글자가 박혔다** -> "글자 없이" 를 다시 요청합니다.

**다시 만들라고 하면 됩니다.** 한 번에 안 나오는 게 정상입니다. 두세 번 고쳐 말하면 나옵니다.

## 5. 저장할 때

| | |
|---|---|
| 파일 형식 | **PNG** (또는 JPG) |
| 파일 이름 | **내 GitHub 아이디** 예: `hong.png` |
| 파일 크기 | **1MB 이하** |

**파일 이름에 실명, 전화번호, 생년월일을 넣지 마세요.** 이 저장소는 누구나 볼 수 있습니다.

**1MB가 넘으면**: 사진 크기를 줄입니다. 맥은 `미리보기`로 열어 `도구` > `크기 조정`, 윈도우는 `그림판`으로 열어 `크기 조정`. 가로세로 **800픽셀** 정도면 충분합니다.

## 6. 어디서 만드나요

이미지를 만들 수 있는 AI면 무엇이든 됩니다. 이미 쓰고 계신 것으로 하세요.
같은 프롬프트를 넣어도 도구마다 결과가 다릅니다. 그것도 오늘 볼 만한 차이입니다.

---

## 왜 이렇게 까다롭게 하나요

혼자 쓰는 그림이면 아무래도 됩니다. **그런데 오늘은 열 명 것이 한 화면에 붙습니다.**

- 어떤 건 세로로 길고 어떤 건 가로로 길면 줄이 안 맞습니다.
- 배경색이 제각각이면 네모가 덕지덕지 보입니다.
- 크기가 10MB면 화면이 느려집니다.

**여럿이 같은 곳에 올릴 때는 형식을 먼저 맞춥니다.** 오늘 배우는 것이 그겁니다.
