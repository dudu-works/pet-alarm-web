# 펫 알람 (pet-alarm-web)

라이브러리 없이 파일 몇 개로 도는 알람 시계입니다. 시각을 맞춰두면 화면 속 펫들이 다 같이 반응합니다.

이 실습은 저장소 하나를 오늘 여기 있는 **모두가 같이** 씁니다. 내 저장소가 따로 있는 게 아니라, 우리 반이 같은 주소 하나를 같이 고칩니다.

| | |
|---|---|
| **화면** | https://pet-alarm-web.vercel.app |
| **저장소** | https://github.com/dudu-works/pet-alarm-web |

**커밋하면 1분 안에 화면에 반영됩니다.** 따로 배포 버튼을 누르지 않아도 됩니다.

펫 사진 만드는 자세한 방법은 같이 받은 `내_펫_만들기` 문서를 보세요. **크기와 배경 형식이 맞아야 벽이 깔끔합니다.**

## 알람 사용법

1. 시각을 고르고 `알람 추가`를 누릅니다.
2. 그 시각이 되면 소리가 나고 펫들이 흔들립니다.
3. `알람 끄기`를 누르면 멈춥니다.

## 내 펫 올리기

### 1) 내 펫 사진 만들기

생성형 AI로 내가 좋아하는 동물 사진을 한 장 만듭니다. 정지 사진이면 됩니다. 움직이지 않아도 됩니다.

프롬프트 예시입니다. 그대로 써도 되고 동물만 바꿔도 됩니다.

- `하얀 배경, 정면을 보고 앉아있는 귀여운 강아지 한 마리, 사진처럼 사실적인 스타일, 정사각형 구도`
- `투명 배경 PNG, 귀여운 고양이 캐릭터 일러스트, 스티커처럼 단순한 선`
- `연한 하늘색 단색 배경, 웃고 있는 아기 판다 얼굴 클로즈업, 정사각형 구도`

배경이 단색이거나 투명하면 화면에 붙였을 때 더 깔끔하게 보입니다.

크기는 정사각형에 가깝게 만듭니다. 파일은 1MB 이하로 저장합니다. 큰 사진은 화면을 느리게 만듭니다.

파일 이름은 내 GitHub 아이디로 합니다. 예를 들어 GitHub 아이디가 hong이면 `hong.png`로 저장합니다. 실명이나 전화번호 같은 개인정보는 파일 이름에 넣지 않습니다.

### 2) 올리기 - 여기가 오늘의 실습

1. 저장소 페이지에서 `Add file` > `Upload files`를 누르고, `pets` 폴더 안에 내 사진을 올립니다.
2. `pets.json` 파일을 엽니다(연필 모양 아이콘을 누르면 고칠 수 있습니다). 마지막 줄 뒤에 콤마를 붙이고, 내 파일 이름을 새 줄로 추가합니다.

예를 들어 지금 `pets.json`이 이렇게 생겼다면

```json
[
  "dudu-works.png"
]
```

내 파일(`hong.png`)을 추가한 뒤에는 이렇게 됩니다.

```json
[
  "dudu-works.png",
  "hong.png"
]
```

3. 커밋합니다.
4. 잠시 뒤 주소를 새로고침하면 내 펫이 벽에 붙어 있습니다.

### 3) 그런데 다 같이 하면

여러 사람이 같은 `pets.json`을 동시에 고치면 나중에 올린 사람 것만 남습니다.
내 이름이 사라졌다면 내가 잘못한 게 아닙니다. 그런 일이 원래 일어납니다.
사라진 줄은 `History`에 그대로 있습니다. 오늘 그걸 찾아서 다시 넣는 걸 해봅니다.

## 폴더 안에 있는 것

- `index.html` - 알람 화면 전체(디자인과 동작 코드가 이 파일 하나에 다 들어있습니다)
- `pets.json` - 펫 벽에 띄울 파일 이름 목록
- `pets/` - 실제 펫 사진들이 들어있는 폴더. 처음에는 강사 사진 한 장(`dudu-works.png`)만 있습니다
